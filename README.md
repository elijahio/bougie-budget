   ScrollReveal.js

- Must have a polished frontend / UI

   https://elijahio.github.io/bougie-budget

- Must meet good quality coding standards (indentation, scoping, naming)

   See for yourself.

   Hah, easy.

   Firebase at https://BougieBudget.net
